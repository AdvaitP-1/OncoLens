# Sample Cases

Demo CSV files for OncoLens wearables pipeline.

- **patient_a_high_priority.csv** – Elevated heart rate and declining SpO2 (high priority)
- **patient_b_needs_review.csv** – Moderate vitals, needs review
- **patient_c_deferred_low_quality.csv** – Low activity, stable vitals (deferred)

Columns: `timestamp`, `heart_rate`, `spo2`, `activity_level`
